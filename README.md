# amusement-park-server
基于SpringBoot3+Vue3前后端分离的游乐园购票系统，系统包含管理员、员工、用户角色，功能如下： 管理员：首页统计；用户管理；员工管理；游乐项目；订单管理（购票订单、项目评价）；新闻资讯（新闻分类、新闻资讯、新闻评论）；轮播管理；系统留言；公告信息；关于本园。 员工：游乐项目；门票计划（管理门票计划和票源）；购票订单（出票、取消）；项目评价；公告信息；个人信息修改；密码修改。 用户：注册；登录；首页；园区介绍；游乐项目（分类、项目列表、详情、收藏、评论、门票信息、购票）；新闻资讯（分类、新闻列表、详情、点赞、评论）；系统留言；通知公告。用户还具有个人中心，功能包括：购票订单、项目评价、项目收藏、新闻点赞、新闻评论、系统留言、个人资料、修改密码。
